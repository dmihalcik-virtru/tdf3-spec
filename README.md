# TDF3 Project Specification

This repo contains the specification for the Trusted Data Format.

## Documentation
* [API](api/) - Swagger templates defining the TDF3 service APIs.
* [Schema](schema/) - Schema definitions for common TDF data objects.
* [Protocol](protocol/) - High-level design of TDF architecture and process workflows.
* [SDK](sdk/) - Documentation for SDK setup and usage.
